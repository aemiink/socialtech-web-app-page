# GLOBAL PAGE RULES

Every sidebar tab page must include:

1. Page Header

* Page title
* Short explanation
* Date range selector or status badge when relevant

2. Main Content

* Service-specific cards, tables, timelines, calendars, boards, charts, or checklists

3. Agency Comment
   Every important page must include a human-written explanation block:
   “Ajans Yorumu”

4. Client Actions
   Where relevant, include:

* Approve
* Request Revision
* Send Missing Information
* Review Report
* Confirm Meeting
* Download File

5. Recent Activity
   Where relevant, include recent changes or timeline entries

Do not use the same generic layout for every page.
Each page must match the operational meaning of that sidebar item.
