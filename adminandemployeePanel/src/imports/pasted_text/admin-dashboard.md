Design a premium dark-theme ADMIN PANEL for a digital growth agency called “Social Tech”.

This is NOT the client panel.
This is the internal management panel used by Social Tech admins and managers to control the agency operation.

The admin panel must help the agency manage:

* clients
* services
* projects
* tasks
* approvals
* campaigns
* contents
* reports
* meetings
* employees
* finance
* automations
* system settings

IMPORTANT:
Do NOT create only one dashboard screen.
Create a complete multi-page admin panel with separate frames/pages for every main sidebar item.

* Dark premium SaaS / agency operations aesthetic
* Background: #131313
* Card surfaces: #1A1A1A / #202020
